二、本图书馆ERP信息管理系统说明

1、建立两个分类：任何图书均同时属于两类。主类和次类，用相同的分类标准。分类尽量客观，也支持交叉查询。

2、为方便查询，特别设立一个关键字。图书分类除了主类和次类以外，每本书还有一个关键字作为第三级分类，增加查询能力。

3、图书综合评分：主要依据作者水平及内容优劣，同时参考对于读者的实用价值。评分可能随时间动态变化。

4、书名：以该图书在版编目（CIP）数据为准。若为译本，采用翻译后的书名。

5、位置：图书存放书架的具体地点，请参考书架示意图。存放位置可能随时调整。

6、语言：该图书内容撰写使用的语言。若有多种语言，以主要或目的语言为准。

7、别名：图书的副标题、第二个名称或简称。若为翻译本，则为原文书名。

8、作者：图书第一作者，尽量采用原文。若为外国作者，一般只用姓氏。若有重复，采用全名。

9、购入时间：购买或拥有该图书的时间。若为借阅，这记录认真阅读（获得）的时间。快速浏览的图书不计入内。

10、出版社：以该图书在版编目（CIP）数据为准。若为盗版或共享版，另作标示。

三、本图书馆分类参考日本的 Nippon Decimal Classification 体系。

日本十进分类法（Nippon Decimal Classification、NDC），是日本参考杜威十进制图书分类法，所发展适用于日本国情的图书分类法。最初的版本为1928年发表的“和洋图书共用十进分类法案”，1929年始改为现名，最新的版本为1995年8月发行的“新订9版”。

The Nippon Decimal Classification (NDC, also called the Nippon Decimal System) is a system of library classification developed for mainly Chinese and Japanese language books maintained by the Japan Library Association since 1956. It is based on the Dewey Decimal System. The system is based upon using each successive digit to divide into nine divisions with the digit zero used for those not belonging to any of the divisions.
